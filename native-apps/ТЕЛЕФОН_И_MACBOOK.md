# Muzic2 на телефоне и MacBook

## MacBook

### Вариант 1: Нативное приложение (рекомендуется)

1. Соберите приложение:
   ```bash
   cd native-apps/macos
   ./build.sh
   ```
2. Запустите:
   ```bash
   open build/Muzic2.app
   ```
   Приложение поднимет локальный PHP-сервер и откроет интерфейс в окне.

### Вариант 2: Браузер

- Запустите MAMP (или другой сервер с PHP).
- Откройте в Safari или Chrome: `http://localhost:8888/muzic2/public/`  
  (порт и путь могут отличаться в зависимости от настроек MAMP).

## Телефон (iPhone / Android)

Чтобы пользоваться Muzic2 с телефона как приложением:

1. **Общая сеть**  
   Телефон и MacBook должны быть в одной Wi‑Fi сети.

2. **Сервер на Mac**  
   На MacBook запустите MAMP и откройте в браузере:  
   `http://localhost:8888/muzic2/public/` — убедитесь, что сайт открывается.

3. **Узнайте IP Mac**  
   На Mac: **Системные настройки → Сеть** или в терминале:
   ```bash
   ipconfig getifaddr en0
   ```
   Например: `192.168.1.5`.

4. **Откройте с телефона**  
   В Safari (iPhone) или Chrome (Android) введите:
   ```
   http://192.168.1.5:8888/muzic2/public/
   ```
   Подставьте свой IP и порт (8888 — стандартный порт MAMP).

5. **Добавить на главный экран (PWA)**  
   - **iPhone:** кнопка «Поделиться» → «На экран „Домой“».  
   - **Android:** меню (три точки) → «Установить приложение» / «Добавить на главный экран».  

   После этого Muzic2 будет открываться как приложение (без адресной строки), с иконкой на главном экране и кэшированием для офлайн-страницы.

### Важно для телефона

- Плеер и навигация адаптированы под касания (крупные зоны нажатия).
- Учтены безопасные зоны экрана (вырез и индикатор на iPhone).
- Для работы с телефона сервер (MAMP) на Mac должен быть запущен и доступен по Wi‑Fi.
