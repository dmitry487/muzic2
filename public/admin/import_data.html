Импорт данных в базу

Запускайте по шагам: сначала треки, потом тексты, потом видео. Источники: tracks/music и tracks/videos.

1) Импорт треков

Добавляет недостающие аудиофайлы в таблицу tracks.

2) Импорт текстов (.lrc)

Привязывает файлы .lrc к трекам в таблицу lyrics.

3) Импорт видео

Привязывает видеофайлы к полю tracks.video_url.

4) Исправить импорт

Исправляет кривые записи: Неизвестный артист, общий альбом Синглы, числовые артисты.

5) Откатить импорт

Удаляет только недавно импортированные треки (по сигнатуре импорта), старые треки не трогает.

Рекомендуется после обновления файлов в папках.
Готово к запуску.
API: /muzic2/public/admin/api/import_data.php
