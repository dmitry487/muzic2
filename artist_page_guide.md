# Страница Артиста - Руководство

## Обзор
Создана страница артиста, которая воспроизводит дизайн из предоставленного изображения (страница Kai Angel в Spotify).

## Созданные файлы:

1. **API Endpoint**: `/public/src/api/artist.php`
   - Возвращает информацию об артисте
   - Топ-треки артиста
   - Альбомы артиста
   - Статистику (общее количество треков, альбомов, продолжительность)

2. **HTML Страница**: `/public/artist.html`
   - Основная структура страницы артиста
   - Hero секция с аватаром и информацией
   - Секция популярных треков
   - Секция альбомов

3. **CSS Стили**: `/public/assets/css/artist.css`
   - Стили в стиле Spotify
   - Responsive дизайн
   - Hover эффекты и анимации

4. **JavaScript**: `/public/assets/js/artist.js`
   - Загрузка данных артиста через API
   - Рендеринг треков и альбомов
   - Интеграция с плеером

## Функциональность:

### Hero Секция:
- Большой круглый аватар артиста
- Имя артиста крупным шрифтом
- Подтверждённый исполнитель (зелёная галочка)
- Количество слушателей в месяц

### Элементы управления:
- Кнопка "Воспроизвести все"
- Кнопка "Перемешать"
- Кнопка "Подписаться/Отписаться"
- Кнопка "Ещё" (опции)

### Популярные треки:
- Нумерованный список треков
- Обложки треков
- Название и статистика
- Кнопки лайка и дополнительных опций
- Продолжительность треков

### Альбомы:
- Сетка альбомов с обложками
- Название и информация об альбоме
- Кнопка воспроизведения при наведении

## Использование:

### Доступ к странице:
```
http://localhost:8888/muzic2/public/artist.html?artist=Kai%20Angel
```

### Из главной страницы:
- Клик по артисту в секции "Артисты" автоматически перенаправит на страницу артиста

### API запрос:
```
GET /public/src/api/artist.php?artist=Kai%20Angel
```

## Дизайн:
- Тёмная тема в стиле Spotify
- Градиентный фон в hero секции
- Размытый фон из обложки артиста
- Зелёный акцентный цвет (#1ed760)
- Адаптивный дизайн для мобильных устройств

## Особенности:
- Анимации при наведении на элементы
- Интеграция с существующим плеером
- Переходы между страницами (артист → альбом)
- Responsive дизайн для всех размеров экранов

## Данные:
Использует существующую базу данных с таблицей `tracks`, группируя данные по артистам для отображения:
- Топ-треки
- Альбомы
- Статистику артиста
