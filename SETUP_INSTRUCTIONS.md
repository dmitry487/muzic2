# Инструкция по запуску Muzic2 на Windows

## Что нужно создать в БД

Скрипт `setup_db.php` автоматически создаст:
- ✅ Базу данных `muzic2` (если её нет)
- ✅ Все необходимые таблицы:
  - `tracks` - треки
  - `artists` - артисты
  - `users` - пользователи
  - `playlists` - плейлисты
  - `likes` - лайки
  - `track_artists` - связи треков с артистами
  - и другие таблицы
- ✅ Начальные данные (треки из schema.sql)

## Что нужно запустить

### ⚠️ ВАЖНО: Сначала запустите MAMP!

1. Откройте приложение **MAMP**
2. Нажмите кнопку **"Start Servers"**
3. Убедитесь, что индикаторы Apache и MySQL зелёные

### Вариант 1: Через батник (САМЫЙ ПРОСТОЙ) ⭐

1. **Дважды кликните на файл:** `setup_db.bat`
2. Скрипт автоматически найдёт PHP и запустит установку
3. Дождитесь сообщения "Установка завершена успешно!"

### Вариант 2: Через командную строку

1. Откройте PowerShell или CMD
2. Перейдите в папку проекта:
   ```powershell
   cd C:\MAMP\htdocs\muzic2
   ```
3. Запустите скрипт установки:
   ```powershell
   C:\MAMP\bin\php\php8.2.0\php.exe scripts\setup_db.php
   ```
   (или найдите вашу версию PHP в `C:\MAMP\bin\php\`)

   Должно появиться:
   ```
   OK
   {
       "success": true,
       "executed": ["schema.sql"],
       "health": {
           "tracks": true,
           "artists": true
       }
   }
   ```

### Вариант 3: Через браузер

1. Убедитесь, что MAMP запущен (Apache зелёный)
2. Откройте в браузере:
   ```
   http://localhost:8888/muzic2/scripts/setup_db.php
   ```
   или
   ```
   http://localhost/muzic2/scripts/setup_db.php
   ```

   Должен вернуться JSON с `"success": true`

## Проверка работы

После запуска setup_db.php:

1. Откройте главную страницу:
   ```
   http://localhost/muzic2/public/index.php
   ```
   или
   ```
   http://localhost:8888/muzic2/public/index.php
   ```

2. Если видите ошибку "Не удалось загрузить данные":
   - Проверьте, что MAMP запущен
   - Проверьте, что MySQL работает (порт 3306 или 8889)
   - Проверьте конфигурацию в `src/config/db.php`:
     - host: `localhost`
     - username: `root`
     - password: `root`
     - dbname: `muzic2`

## Если что-то не работает

### Ошибка "Не удалось подключиться к MySQL"

Если видите эту ошибку:

1. **Проверьте MAMP:**
   - Откройте MAMP
   - Убедитесь, что кнопка "Start Servers" нажата
   - Индикатор MySQL должен быть **зелёным**
   - Если красный - нажмите "Stop Servers", затем "Start Servers" снова

2. **Проверьте порт MySQL:**
   - В MAMP нажмите "Preferences" → "Ports"
   - Запомните порт MySQL (обычно 3306 или 8889)
   - Если порт другой, обновите `src/config/db.php`

3. **Запустите проверку подключения:**
   - Дважды кликните на `check_mysql.bat`
   - Скрипт покажет, к какому порту можно подключиться

4. **Проверьте пароль MySQL:**
   - Откройте phpMyAdmin: http://localhost:8888/phpMyAdmin
   - Попробуйте войти с логином `root` и паролем `root`
   - Если пароль другой, обновите `src/config/db.php`

5. **Проверьте, создана ли БД:**
   - В phpMyAdmin должна быть база `muzic2`
   - Если её нет, скрипт создаст её автоматически

6. **Проверьте таблицы:**
   - В phpMyAdmin откройте базу `muzic2`
   - Должны быть таблицы: `tracks`, `artists`, `users` и др.

7. **Проверьте данные:**
   ```sql
   SELECT COUNT(*) FROM tracks;
   ```
   Должно быть больше 0.

## Важные файлы

- `scripts/setup_db.php` - скрипт создания БД и таблиц
- `db/schema.sql` - схема БД и начальные данные
- `src/config/db.php` - конфигурация подключения к БД
- `src/config/db_windows.php` - Windows-специфичная конфигурация
