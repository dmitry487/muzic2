# iOS приложение Muzic2

## Установка и настройка

1. **Откройте проект в Xcode:**
   ```bash
   open Muzic2App/Muzic2App.xcodeproj
   ```

2. **Скопируйте контент:**
   - Из корня проекта запустите: `npm run copy-content` (из папки app)
   - Или вручную скопируйте папку `app/content/` в `ios/Muzic2App/Muzic2App/content/`

3. **Добавьте контент в Xcode:**
   - Перетащите папку `content/` в проект Xcode
   - Выберите "Create folder references" (не "Create groups")
   - Убедитесь, что папка добавлена в Target Membership

4. **Настройте Bundle Identifier:**
   - Откройте проект в Xcode
   - Выберите проект в навигаторе
   - В разделе "Signing & Capabilities" укажите свой Team

5. **Соберите и запустите:**
   - Выберите симулятор или устройство
   - Нажмите Cmd+R для запуска

## Особенности

- Использует WKWebView для отображения веб-контента
- Все файлы включены в bundle приложения
- Работает полностью офлайн
- Поддерживает все функции веб-версии

## Требования

- Xcode 14.0 или новее
- iOS 14.0 или новее
- Swift 5.0
